> 🚧 **Building in public:** Follow this project from initial planning through SQL development, analytics, and Version 1.0.

---

## Project Overview

LTC Medicaid Analytics is a portfolio project inspired by real-world long-term care Medicaid workflows and healthcare reimbursement operations.

The project simulates the analytical process used by healthcare organizations to monitor Medicaid applications, resident admissions, reimbursement timelines, payer mix, financial performance, and operational efficiency.

From planning and database design through SQL development, reporting, dashboard creation, and business recommendations, this repository demonstrates the complete lifecycle of an analytics project.

---

## Business Problem

Long-term care organizations manage hundreds of residents while tracking complex Medicaid eligibility, reimbursement timelines, payer transitions, and financial risk.

Without reliable reporting, organizations may struggle to:

- Monitor Medicaid Pending balances
- Track application progress
- Identify reimbursement delays
- Analyze payer mix
- Measure operational performance
- Reduce revenue at risk

This project demonstrates how SQL can transform operational healthcare data into meaningful business intelligence.
